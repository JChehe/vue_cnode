由于经验不足，若各位大大发现应用或代码哪里有可以改进或不完善的地方，欢迎大家在提 issue 或 pr，也可以评论。

### 本地部署
当然，你也可以在本地部署该项目。
``` bash
# install dependencies
npm install

# serve with hot reload at localhost:8080
npm run dev

# build for production with minification
npm run build
```
